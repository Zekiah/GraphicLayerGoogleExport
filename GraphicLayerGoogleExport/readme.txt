Installation

You will need to download the latest version of the ESRI Silverlight API at 
http://help.arcgis.com/en/webapi/silverlight/
After downloading, make sure to include a reference to the following libraries.

ESRI.ArcGIS.Client
ESRI.ArcGIS.Client.Toolkit
